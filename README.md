# hdd_failure_prediction

Учитывая данные мониторинга состояния диска S.M.A.R.T (https://ru.wikipedia.org/wiki/S.M.A.R.T.) и данных о неисправностей, необходимо определить, выйдет ли из строя каждый диск в течение следующих 30 дней.
